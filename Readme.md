### Fuzzy Questions App ###
This is a simple app that allows to create questions with multiple answers with fuzzy logic

### How to run the app ###
1. Clone the repository
2. Run the command
```bash
make up
```
3. Open the browser and go to the address
```
http://127.0.0.1
```

### How to run the tests ###

1. Run the command
```bash
make test
```
